# MD5 HashSolver
A bruteforce python MD5 hasher

## Features

 **Hash Solver**
 - Variable length hexadecimal string unhashing
 - Hashrate per second printing 
 - Attempted Hashes
 

**Hash Creator**
 - Create variable length MD5 hash
